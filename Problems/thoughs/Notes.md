## XOR
- A XOR B = C => C XOR A = B
- - (PARTIAL XOR) XOR (MISSING ELEMENT) = (TOTAL XOR) => (TOTAL XOR) XOR (PATIAL XOR) = (MISSING ELEMNT)

## MODULES
( a + b) % c = ( ( a % c ) + ( b % c ) ) % c
( a * b) % c = ( ( a % c ) * ( b % c ) ) % c
( a – b) % c = ( ( a % c ) – ( b % c ) ) % c
